PRE-DEMO SETUP (do this 1 minute before recording)
---------------------------------------------------
1. Open https://bharat-kavach.vercel.app in your browser
2. Wait 30 seconds for Render backend to wake up
3. Zoom browser to 80% so the full dashboard fits on screen
4. Start screen recording (Win+G on Windows)

-------------------------------------

I am calling from the CBI office regarding a serious money

14 illegal bank accounts. You are now under digital arrest. Do
not disconnect this call or inform your family. To avoid

"Our BehavioralClassifier — powered by Google Gemini — maps

an embedded India SVG. Cases are tagged by city and grouped —
giving command centres a live geospatial intelligence layer."

[1:50 – 2:15] FEATURE 3 — KILL SWITCH INTERVENTION
-----------------------------------------------------
SCREEN: Point to the red AUTOMATIC INTERVENTION TRIGGERED banner

SAY:
"When risk exceeds 85% AND the Financial Extraction stage is

Three protective actions execute simultaneously:

webhook calls to bank UPI APIs, TRAI telecom gateway, and

SCREEN: Scroll to Currency Verification, upload any currency JPEG

SAY:

IMPLEMENTATION: Calibrated against 215 real currency images.
Precision: 86.7%. The model flags suspicious notes only when
BOTH edge density AND sharpness signals are below threshold —
avoiding false alarms on low-quality genuine note photos."

DEMO NOTES

MAP HOTSPOTS: Each transcript submission now auto-assigns a
random Indian city (Delhi, Mumbai, Bangalore, Chennai, Kolkata,
Hyderabad, Pune, Ahmedabad). Submit 2-3 transcripts before
recording and the map will show multiple coloured hotspots.

BRAVE BROWSER: No INP warnings anymore — the fix is live.
Use any browser you prefer.

DOCUMENT FORENSICS: Upload a JPEG (not PDF) for fastest results.
Analysis takes 20-40 seconds — Gemini Vision needs processing time.

RENDER WAKEUP: If the first ANALYZE call is slow (30+ seconds),
it's Render waking up. Wait for it. Subsequent calls are fast.

CACHE: If you submit the same transcript twice, it serves from
cache instantly — useful if you need to redo a demo take.
